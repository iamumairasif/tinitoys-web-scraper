# TiniToys Web Scraper

## Overview
This project is a Python web scraper developed to extract product information from the TiniToys e-commerce website using Requests and BeautifulSoup.

## Features
- Extracts product names
- Extracts product prices
- Extracts product codes
- Collects product URLs
- Scrapes multiple categories and pages
- Saves data into a CSV file

## Technologies
- Python
- Requests
- BeautifulSoup
- Pandas

## Output
The scraper successfully extracted over 608 product records and exported them into a structured CSV file.
